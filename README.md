
# PH Man 

## Requerimientos
+ Docker
+ Docker-Compose

## Instalacion y Configuracion
+ Correr los container
```sh
docker-compose up -d
```
- *localhost:3000*  Server Node,
- *localhost: 5630*  Postgres,
- *localhost: 5050*  PgAdmin4

+ Configurar PgAdmin4
    Añadir un nuevo servidor
